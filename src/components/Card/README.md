卡片

## API

### Card

| 属性        | 说明                                 | 类型                  | 默认值  |
| ----------- | ------------------------------------ | --------------------- | ------- |
| title       | 标题                                 | `ReactNode`           | -       |
| hasColorBar | 是否带有主题条                       | `boolean`             | `true`  |
| hasBorder   | 是否有边框                           | `boolean`             | `false` |
| hoverable   | 是否有悬浮阴影效果，仅在有边框下触发 | boolean               | `false` |
| footer      | 底部渲染                             | `ReactNode`           | -       |
| footerClass | 底部 css 类名                        | `string`              | -       |
| footerStyle | 底部 style                           | `React.CSSProperties` | -       |
| 其它属性    | 所有 div 原生属性                    | -                     | -       |
